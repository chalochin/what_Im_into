## Customization

To personalize this page:

1. **Update the Profile Image**
   - Replace the placeholder image URL in line 82 with your own image URL
   - Use services like Imgur, Cloudinary, or host directly on GitHub

2. **Update Social Links**
   - Replace the `href="#"` values with your actual social media URLs
   - Add or remove links as needed

3. **Change Colors**
   - Modify the background color in the CSS (line 12)
   - Adjust the purple theme color in the `.w3-purple` class (lines 68-71)

## Deployment

This page can be deployed to:
- **GitHub Pages** - Free hosting directly from this repository
- **Netlify** - Easy drag-and-drop deployment
- **Vercel** - Simple and fast deployment
- **Any static web host** - Works anywhere that serves HTML

### GitHub Pages Setup

1. Go to repository Settings → Pages
2. Set the source to the `master` branch
3. Your page will be live at `https://chalochin.github.io/what_Im_into/`
